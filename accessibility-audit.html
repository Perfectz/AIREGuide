üß™ Manual Testing Instructions

1. Keyboard Navigation Test

  • Use Tab key to navigate through all interactive elements
  • Use Shift+Tab to navigate backwards
  • Use Enter/Space to activate buttons and links
  • Use Escape key to close mobile menu

2. Screen Reader Test

  • Use NVDA, JAWS, or VoiceOver to test screen reader compatibility
  • Verify all interactive elements are announced properly
  • Test mobile menu announcements
  • Verify skip links work correctly

3. Mobile Device Test

  • Test on actual mobile devices (not just browser simulation)
  • Verify touch targets are large enough (44px minimum)
  • Test mobile menu functionality
  • Verify responsive design works across different screen sizes

4. Color and Contrast Test

  • Use browser dev tools to test contrast ratios
  • Test with high contrast mode enabled
  • Verify focus indicators are visible
  • Test with color blindness simulators
